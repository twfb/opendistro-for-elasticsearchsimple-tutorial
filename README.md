## Options
You can choose to install one of "elasticsearch-sql" and "opendistro-for-elasticsearch".

## Install
### elasticsearch-sql
```
cd elasticsearch-sql
./build
./start
cd tutorial
./init_data
./login
```
###  opendistro-for-elasticsearch 
```
cd opendistro-for-elasticsearch 
./start
cd tutotial
./init_data
./login
```
